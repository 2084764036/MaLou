# MaLou小程序 - 高保真文字版原型（含跳转流程图）

说明：原型按“用户操作流程”排序，明确每个页面的尺寸适配（微信小程序标准尺寸：375px宽）、元素位置、交互逻辑，所有按钮/组件均标注点击后跳转路径，开发/设计可直接对照落地，贴合之前的PRD需求，无额外冗余功能；每个页面末尾补充跳转流程图，清晰呈现页面间关联。

# 一、基础设置（全局通用）

- 导航栏：所有页面统一导航栏（高度44px），背景色#ffffff，文字#333333，左侧无返回键（除子页面），右侧无额外按钮

- 底部Tab栏：固定在页面底部（高度50px），共3个Tab，图标+文字，默认选中“首页”，未选中#999999，选中#3498db

- Tab栏内容：首页 → 圈子广场 → 我的

- 加载状态：所有跳转/数据加载时，显示圆形加载动画（居中，浅灰色，无文字）

- 提示语：操作成功/失败时，顶部弹出Toast（高度40px，居中，背景#333333，文字#ffffff，显示2秒后消失）

# 二、页面详细原型（按用户流程排序）

## 页面1：启动页（首次进入）

- 整体布局：全屏白色背景，无多余元素，居中展示核心内容

- 顶部：距离顶部100px，展示小程序Logo（圆形，直径80px，底色#3498db，中间“MaLou”字样白色）

- 中间：Logo下方20px，展示文字“MaLou”（字体24px，#333333，加粗）

- 中间下方：文字下方10px，展示副标题“全场景组队，找搭子不孤单”（字体14px，#666666）

- 底部：距离底部80px，展示“微信一键登录”按钮（宽度280px，高度48px，圆角8px，背景#3498db，文字#ffffff，字体16px）

- 交互逻辑：点击“微信一键登录” → 触发微信授权弹窗 → 授权成功 → 跳转至首页；授权失败 → 弹出Toast“授权失败，请重新尝试”

- 跳转流程图：启动页 → （微信授权成功）→ 首页；启动页 → （微信授权失败）→ 停留启动页（弹出失败提示）

# 三、补充说明（开发/设计重点）

- 适配：所有页面适配微信小程序所有机型，禁止出现横向滚动条，文字不溢出

- 颜色：主色#3498db，辅助色#52c41a（成功）、#ff4d4f（错误）、#faad14（警告），背景色#f5f5f5、#ffffff，文字色#333333、#666666、#999999

- 字体：统一使用微信默认字体（PingFang SC），标题18px，正文14px，辅助文字12px、11px

- 交互：所有可点击元素（按钮、卡片、图标），点击时有轻微按压效果（背景色变浅/边框变色）

- 异常处理：无网络时，显示“无网络，请检查网络设置”提示；无数据时，显示“暂无相关内容”提示（居中，文字#999999）
